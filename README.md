coriander-bot-rs
======

烧饼 Telegram Bot 的重构版本

# 运行

1. 将项目克隆至本地
2. 设置环境变量 `TELOXIDE_TOKEN` 为 Telegram Bot Token
3. (可选) 设置环境变量 `TELOXIDE_PROXY` 为本地代理
4. 执行以下命令
   ```shell
   cargo run --package coriander-bot-rs --bin coriander-bot-rs
   ```
